7.3.5 Congruence Modulo ùíè and Congruence Classes

In Beginning Activity 2, we used the notation C‚Å¢[k] for the set of all integers that are congruent to k modulo 3. We could have used a similar notation for equivalence classes, and this would have been perfectly acceptable. We have seen that congruence modulo 3 divides the integers into three distinct congruence classes. Each congruence class consists of those integers with the same remainder when divided by 3. In a similar manner, if we use congruence modulo 2, we simply divide the integers into two classes. 7.3.9 Partitions and Equivalence Relations

A partition of a set A is a collection of subsets of A that "breaks up" the set A into disjoint subsets. Technically, each pair of distinct subsets in the collection must be disjoint. We then say that the collection of subsets is pairwise disjoint. We introduce the following formal definition.

Definition. Let A be a nonempty set, and let ùíû be a collection of subsets of A. The collection of subsets ùíû is a partition of ùë® provided that

  1. 1.

    For each V‚ààùíû,V‚â†‚àÖ.

  2. 2.

    For each x‚ààA, there exists a V‚ààùíû such that x‚ààV.

  3. 3.

    For every V,W‚ààùíû,V=W or V‚à©W=‚àÖ.

There is a close relation between partitions and equivalence classes since the equivalence classes of an equivalence relation form a partition of the underlying set, as will be proven in Theorem 7.18. The proof of this theorem relies on the results in Theorem 7.14.

Theorem 7.18. Let ‚àº be an equivalence relation on the nonempty set A. Then the collection ùíû of all equivalence classes determined by ‚àº is a partition of the set A.

Proof. Let ‚àº be an equivalence relation on the nonempty set A, and let ùíû be the collection of all equivalence classes determined by ‚àº. That is,

ùíû={[a]‚à£a‚ààA}.

We will use Theorem 7.14. to prove that ùíû is a partition of A.
Part (1) of Theorem 7.14 states that for each a‚ààA,a‚àà[a]. In terms of the equivalence classes, this means that each equivalence class is nonempty since each element of A is in its own equivalence class. Consequently, ùíû, the collection of all equivalence classes determined by ‚àº, satisfies the first two conditions of the definition of a partition.

We must now show that the collection ùíû of all equivalence classes determined by ‚àº satisfies the third condition for being a partition. That is, we need to show that any two equivalence classes are either equal or are disjoint. However, this is exactly the result in Part (3) of Theorem 7.14.

Hence, we have proven that the collection ùíû of all equivalence classes determined by ‚àº is a partition of the set A.

Note: Theorem 7.18 has shown us that if ‚àº is an equivalence relation on a nonempty set A, then the collection of the equivalence classes determined by ‚àº form a partition of the set A.

This process can be reversed. This means that given a partition ùíû of a nonempty set A, we can define an equivalence relation on A whose equivalence classes are precisely the subsets of A that form the partition. This will be explored in Exercise (12).
