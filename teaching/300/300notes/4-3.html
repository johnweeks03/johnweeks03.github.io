Mathematics begins to defy our expectations when we start dealing with infinite numbers of sets. Truthfully, we do not have a good grasp of what an infinite number of objects looks like. There are not an infinite number of atoms in our universe, not an infinite number of seconds in our universe’s history, and not an infinite number of possibilities that can take place in any billions of years beyond this point in time.

But there is nothing better equipped to handle the infinite than mathematics. Indeed, “limits at infinity” help us approximate end behavior of functions by describing what happens as x grows infinitely large or infinitely more negative. We can calculate the sum of an infinite number of terms, which we call an infinite series. We can determine which sequences of infinitely many numbers converge and which diverge. We even use an infinite number of functions added together to define a function using Taylor series and Maclaurin Series.

Why do we not like these topics very much? Perhaps it is exactly because of this notion of infinity that we don’t understand very well. But these very topics are why mathematics is incredibly useful in computing huge amounts of data quickly and accurately. It is why we can be sure our actuarial methods are solid no matter how big the accounts, no matter how long we use them. We can do this all day… for an infinite amount of time… and mathematics will still hold firm.

For these reasons we dare to breach the infinite, just like we did in the previous example. Let’s make rigorous this notion of “infinite union” and “infinite intersection”.

Definition 4.3.2.

An index set Δ is an arbitrary collection of elements where each element i∈Δ has a set Ai defined for that element.

Let {Aα}α∈Δ be a collection of sets for some index set Δ. Then the union of the entire collection of sets {Aα}α∈Δ is defined to be

⋃α∈ΔAα:={x∣x∈Aα⁢ for at least one ⁢α∈Δ}.

We define the intersection of the entire collection of sets {Aα}α∈Δ to be

⋂α∈ΔAα:={x∣x∈Aα⁢ for all ⁢α∈Δ}.

In the special case that Δ=ℕ, we write

⋃n=1∞An={x∣x∈An⁢ for at least one ⁢n∈ℕ}

and

⋂n=1∞An:={x∣x∈Aα⁢ for all ⁢n∈ℕ}.

The LaTeX for these symbols: ⋃ is \bigcup, and ⋂ is \bigcap.
