5.1.2 Well-Defined Functions

(Definition)5.1.3.

Let X and Y be two nonempty sets. A function f from X to Y is a rule from X to Y that assigns to every element x∈X a unique element y∈Y. We usually will write f⁢(x)=y (read “f of x equals y”) to describe that the function f maps x to y.

The set X is called the domain of f, and the set Y is called the codomain of f.

An important subset of the codomain is the range of f, defined as

f⁢(X):={y∈Y∣(∃x∈X)⁢f⁢(x)=y}.

In the expression “f⁢(x)=y”, x is considered the input of the function, with y being the corresponding output of the function.

Note that we define a function as a “rule”. A natural question might be, “What is a rule?” For now we will understanding what this means intuitively - later we will define a function as a special case of relation (Chapter 7).

The last example from the previous page is an example of a function that is not well-defined. If there are multiple different ways to express a value of an input, the output must be the same irrespective of the expression. For example, even though 1/2=2/4, s⁢(1/2)≠s⁢(2/4), so technically s cannot be a function as the same value is being mapped to two different outputs.
